## 时间序列

```
using TimeSeries
```

新建Date变量
```
dates = collect(Date(2018, 1,1) :Day(1): Date(2018, 12, 31))
dates[1:20]
```

新建TimeArray
```
ta = TimeArray(dates, rand(length(dates),2))
```
